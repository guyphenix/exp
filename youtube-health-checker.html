YouTube Kids
Health Checker

Real channel analysis using YouTube Data API v3. Public metrics, genuine intelligence, honest about what it can and can't see.

🔑
YouTube API Key Required

Get a free key in ~3 minutes: console.cloud.google.com → New Project → Enable YouTube Data API v3 → Credentials → Create API Key. Free quota: 10,000 units/day (enough for ~200 channel checks).

Accepts: @handles, full URLs, or channel IDs starting with UC

What this tool actually measures

  • Subscriber count & total views
  • Upload frequency & cadence
  • Per-video view counts & engagement
  • Made for Kids status (COPPA)
  • Channel age & growth trajectory
  • Average views per video
  • RPM / ad revenue (owner-only)
  • Watch duration / retention (owner-only)
  • Click-through rate (owner-only)
  • Impressions (owner-only)
